Data Structures and Loops

Data Structures

In the previous class, we covered primitive data types, like numbers, strings, and booleans. But JavaScript also provides more complex ways of storing and manipulating data, called data structures.

These data structures are a kind of "compound" data type, combining multiple values into a single unit. Different data structures serve different purposes and have unique characteristics.

Arrays

Arrays, or lists, are ordered collections of values. Each value in an array is called an element, and each element has a unique index that can be used to access it.

Arrays are useful for storing and manipulating collections of data, such as lists of names, numbers, or objects:

const fruits = ['apple', 'banana', 'cherry'];
console.log(fruits[0]); // Output: 'apple'

Array elements can be reassigned, like normal variables:

fruits[0] = 'grape';
console.log(fruits[0]); // Output: 'grape'

Objects

Objects are collections of key-value pairs. Each key is a string (or symbol), and each value can be any data type, including other objects or functions.

const user = {
    name: 'Tim',
    age: 25,
    email: 'tim@example.com'
};

console.log(user.name); // Output: 'Tim'

As seen above, object properties can be accessed using dot notation.

Loops

Loops are a fundamental programming construct that allow us to repeat a block of code multiple times. They are essential for working with data structures, as they enable us to iterate over arrays and objects.

There are several types of loops in JavaScript, including for loops, while loops, and do while loops.

For Loops

For loops are used to iterate over arrays or execute a block of code a specific number of times.

for (let i = 0; i < 5; i++) {
    console.log(i);
}

In the above example, the for loop iterates 5 times, logging the value of `i` to the console.

for (let i = 0; i < fruits.length; i++) {
    console.log(fruits[i]);
}

In this example, the for loop iterates over the fruits array and logs each fruit to the console.

For of Loops

For of loops are a variant of the for loop used specifically to iterate over the values of an iterable object, such as an array.

for (const fruit of fruits) {
    console.log(fruit);
}

This is functionally the same as the previous example, but with a more concise syntax.

While Loops

While loops execute a block of code as long as a specified condition is true.

let count = 0;
while (count < 5) {
    console.log(count);
    count++;
}

In this example, the while loop continues to execute as long as `count` is less than 5, logging the value of `count` and then incrementing it.

While loops are often useful when the number of necessary iterations is not known in advance.

while(true) {
    // This loop will run indefinitely
    if(condition) {
        break; // Exit the loop if a certain condition is met
    }
}

Do While Loops

Do while loops are similar to while loops, but they execute the block of code at least once before checking the condition.

let count = 6;
do {
    console.log(count);
    count++;
} while (count < 5);

In the above example, the do while loop will execute the block of code at least once, then continue executing as long as `count` is less than 5. Since `count` is initialized to 6, the loop will only execute once before the condition is checked.

Classes

Classes are a way to define 'blueprints' for creating objects in JavaScript. They provide a convenient syntax for creating objects and managing their properties and methods.

We use classes when we need to create multiple objects with similar properties and methods.

class PlayingCard {
    constructor(suit, rank) {
        this.suit = suit;
        this.rank = rank;
    }
}

In this example, we define a class called `PlayingCard` with a constructor that takes two parameters: `suit` and `rank`. The constructor initializes the properties of the object when it is created.

We can then create instances of the `PlayingCard` class by calling the constructor with specific values for `suit` and `rank`:

const card1 = new PlayingCard('hearts', 'A');
const card2 = new PlayingCard('spades', '10');
console.log(card1.suit); // Output: 'hearts'
console.log(card2.rank); // Output: '10'
